1-计算机基础知识

半导体存储器：

  • 所说的内存就是这种，但还有其他
  • 可分为：随机存取存储器（RAM）、只读存储器（ROM）
  • 速度快，体积小

存储器分为：内存、外寸

  • 内存分为：RAM（主存）、ROM
  • 外存：硬盘、磁盘等
  • 主存 != 内存，内存 > 主存
  • 容量的单位为字节数

进制转换问题（别想的太繁杂，记住主要原则，学会类比）

  • X转2进制
    • 10转2进制
      • 法1: 整数除2逆向取余，小数乘2正向取整（小数时不好结束，取几位即可
      • 法2: 自己想的8421-3210法，如13=8+4+1=3+2+0，即1101
    • 8转2进制：8的每位用2的3位表示
    • 16转2进制：16的每位用2的4位表示
  • 2进制转X
    • 2转10：自己的8421-3210法，熟记2的n次幂，1、2、4、8、16、32、64、128（X转10也是此思想）
    • 2转8：每3位一组
    • 2转16：每4位一组
  • 因此：可将2进制看作中介

数据与指令

  • 本质上是相同的，都以二进制的形式存放在计算机中；

  • 二者功能不同：指令用于指导计算机去做什么以及怎样做，而数据是计算机操作的具体对象；

  • 也可以根据访的时机来区分：二者都存储在存储器中，在取指令周期内取出的是指令，同理，在读数据周期内取出的为数据。

编程语言分类

  • 在面向机器或面向用户的角度，可分为三大类，即机器语言、汇编语言以及高级语言；
  • 高级语言中有C/C++、Java、python、PHP 等，若有需要，也可进行细分，比如 CC++更面向底层，而 PHP 或 python 等对底层进行了进一步的封装，可以说更加面向用户。
  • 在程序执行的角度，可分为编译型语言和解释型语言。

编译型与解释型语言

  • 编译型语言：在程序执行之前，源代码就翻译成机器指令，由于事先翻译了，所有执行时速度快，如 C/C++
  • 解释性语言：程序运行之前，不会事先翻译成机器指令，而是一边翻译一边执行，因此，执行速度相对较慢，如 python

其他

  • 数据，以补码形式存在
  • 冯诺伊曼：出入控运存

2-计算机中的数

字节与字长

  • 字长：CPU 一次可处理的数据的最大长度，其位数取決于具体的计算机
  • 字节：衡量数据量以及存储容量的基本单位，1 字节等于 8 位二进制信息

表示范围

  • 为何n位有符号数的范围负数比正数多一个？如：
    • 8位有符号数的取值范围是-128 ~ +127
    • 16位有符号数的取值范围是-32768 ~ +32767
  • n位有符号数表示范围：-2^(n-1) ~ 2^(n-1)-1，暂不深究原理（意义不大且麻烦
  • n位无符号数表示范围：0 ~ 2^n - 1，如8位无符号范围0～255
  • 1GB = 1024MB,1MB=1024KB,1KB=1024B,1B =8bit

5-运算符基础

  • /运算符：整数相除，结果也为整数，要想得小数，必须有浮点数的参与；
  • %运算符的两个操作数：必须是整数，不能是浮点数；
  • ++运算符的操作数不是只能为整数，也可为其他，如浮点数；
  • 数字形式的8-3 = 5，字符形式的'8'-'3' = 5，二者都等于5，对于字符形式来说，完全可以按照数字形式的来计算，因为字符背后也对应着数字，况且也挨着，如‘8’为56，‘3’为51；

逗号运算符

  • 逗号运算符考点

  • 逗号与++运算符

  • 语句：若干运算符与数据组成的算式

  • 表达式：

    • 程序的基本组成
    • 每条语句后面有一个分号；
    • 表达式加分号便可得到一个语句，或者表达式是语句的一部分；
    • 多个语句通过{}扩起来成为复合语句，即代码块

  • sizeof运算符

    • 得到某一数据或数据类型所占内存空间的大小
    • 以字节为单位
    • 参数可以是某个变量名，也可直接是某数据类型

  • 自增自减运算符，若在自增/减时同时还参与运算：

    • 在前：先自增/减，再参与运算；
    • 在后：同上反

自动（隐形）类型转换

  • 算术运算：

    • 不同数据类型的数据在进行运算时，需转为同一种数据类型后才能开始运算；
    • 转换规则：以该算术表达式中，最长类型为目标
    • 即：char->short->int->long->float->double

  • 赋值运算：

    • 右边类型自动转换为左边类型，有可能会丢失数据
    • 如int a = 1.2; //a 为 1；

7-一维数组

  • 数组长度、下标
  • 定义时：长度只可为常量
    • 调用时：下标可常量，可变量
  • 数组定义、存储方式
    • 相同数据类型的变量的集合
    • 其在内存中的存储是连续的
  • 数组定义：数据类型 数组名[数组长度]
  • 数组初始化
    • 整体赋值：a[3] = {1,2,3}
    • 部分赋值：a[3] = {1,2}（剩余元素补0）
    • 灵活赋值：a[] = {1,2,3}（不给长度，根据给的值灵活分配）
  • 数组初始化与赋值
    • 初始化：定义时就给值，称初始化
    • 赋值：定义之后才给值，称赋值（赋值名词的使用无需太严格）

8-字符数组与字符串处理函数

  • 字符数组的赋值

    • 除了可以同普通数组一样进行赋值外
  • 还可以通过字符串来行赋值
    • 字符串最后会有一个\0，因此字符串形式的赋值会比普通形式的赋值默认多一个元素，用来存储\0
    • 宽字节的字符数组，若用字符串来赋值，要加L前缀，如wchar_t wA[]= L"hello";

10-选择结构if

  • 三目运算符

    • 条件 ? 结果1 : 结果2
    • 条件为真，执行1，否则执行2
    • 为真即不为0，>0、<0均可

  • 优先级比较：括号 > 逻辑非 > 算术运算符 > 关系运算符 > 逻辑与 > 逻辑或（不必刻意记忆，了解便可

  • 三角形相关

    • 构成条件：任意两边之和大于第三边：(a+b)>c && (a+c)>b && (b+c)>a
    • 等边：a==b && a==c
    • 等腰：a==b||a==c||b==c

  • 常见ascii码范围

  • 判断某字符是大写字母or小写字母or数字，c >= 'a' && c <= 'z'，直接比较即可，不用记具体ascii码（因为本质都是比较ascii）

  • scanf_s安全版输入函数需要第三个参数：用于限定输入的最大长度

    • 如scanf_s("%c", &c, 1);
    • 但是输入整型时，不需要，仅scanf_s("%d", &n);即可
    • 也验证了此类安全函数是为字符的输入准备的，以免越界现象

  • getchar函数

    • 用于获取单个字符
    • 常用于消化字符，如键入某字符后的回车键
    • 有返回值（返回ascii码），可用字符变量或整型变量接受

  • 闰年：能被4整除但是不能被100整除，或者能被400整除的年份

  • 三个数字排序（不使用数组和循环

  • 判断某整数的位数：通过除运算

  • 思路

    | a | a /= 10 | 位数count | 余数 |

  • 要灵活运用取余和除运算，有大用处！！！

  • printf输出函数不是必须需要格式控制符，可以直接传入字符串打印，如printf("123");（此时传入的是地址，传入数组名时也是传地址，传普通变量名是传值

  • C语言中，1<a<10不可，只可a>1 && a<10，不能写作一个

  • char c =0是正确的，因为字符本质上也是数字
  • case语句中定义变量需要添加大括号（case语句最好也要用大括号括，万无一失）

11-选择结构switch

  • switch-case结构只能基于整数值进行，而不能是浮点型
  • default与case的顺序无关紧要，不影响结果
  • switch结构能实现的，if结构也能实现，反之不然。

适应情况

  • if结构适应情况
    • 逻辑复杂
    • 分支较小
  • switch结构适应情况
    • 逻辑简单
    • 分支较多

优缺点

  • switch缺点
    • 只能基于整数值来进行匹配，不可浮点数
    • 只能判断是否相等，不可基于某区间范围来判断
    • switch可实现的if都能实现，反之不然，即存在一定限制
  • if缺点
    • 分支较多时用if会显得很乱
    • 多层if嵌套也会使得程序变得复杂
