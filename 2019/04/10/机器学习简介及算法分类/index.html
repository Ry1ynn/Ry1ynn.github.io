机器学习简介及算法分类

发表于 2019-04-10 | 分类于 机器学习

教程地址

  • 机器学习是什么？
    • 一帮计算机科学家、
    • 想让计算机像人一样思考、
    • 所研发出来的计算机理论。
  • 思想：人与计算机并无差别
    • 都是一大批、
    • 互相连接的、
    • 信息传递和信息存储元素、
    • 所组成的系统
  • 跨学科的交融，包含统计学、概率论等
  • 随着硬件水平提高，计算机速度提升，逐渐进入视线
  • 常见应用：
    • 谷歌的goole now、Google photo
    • 百度的图片识别，应用了机器学习的视觉处理系统
    • 金融公司的汇率预测、股票涨跌，房地产的房价预测
  • 机器学习不仅一种方法，实现其的方法多种多样（方法即算法

  • 目前所有的机器学习算法大概分为4-5类

    • 有数据和标签的——监督学习
    • 只有数据、没有标签的——非监督学习
    • 结合监督和非监督的——半监督
    • 从经验中总结提升的——强化学习
    • 类似强化学习、适者生存原则的——遗传算法

  • 监督学习，supervise learning

    • 在学习过程中，向计算机提供数据和其对应的值
    • 比如给计算机看猫、狗的图片，并告诉他哪些图片是猫，哪些是狗；然后让他学习去分辨猫狗
    • 通过这种指引的方式，让计算机学习”我们是如何把这些图片数据对应上图片所代表的物体”
    • 也就是让计算机学习”这些标签可以代表那些图片”
    • 预测房屋价格、股票涨停，也可用监督学习来实现
    • 所熟知的神经网络也是一种监督学习的方式
  • 非监督学习，un-supervise learning
    • 在学习过程中，只给计算机提供猫和狗的图片
    • 但是，并无告诉他哪些是狗，哪些是猫
    • 而是让他自己去判断和分类
    • 让其自己总结出两种图片的不同之处
    • 即在学习过程中，不提供数据所对应的标签信息
    • 计算机通过观察各种数据之间的特性，自己发现特性背后的规律
    • 这些规律即非监督学习学到的东西
  • 半监督学习，semi-supervise learning
    • 综合监督和非监督的特征
    • 主要考虑：如何利用少量有标签的样本 和 大量没有标签的样本进行训练和分类

  • 强化学习，reinforcement learning

    • 在规划机器人的行为准则方面，的一种机器学习方法
    • 把计算机丢到一个对他完全陌生的环境，或者让它完成一项从未接触过的任务
    • 他自己会去尝试各种手段，最后让自己成功适应这一陌生的环境，或者学会完成这件任务的方法途径
    • 比如训练机器人投篮，只需要给他一个球
    • 并告知投进记一分，让其自己去尝试各种各样的投篮方式
    • 开始阶段，命中率可能会很低
    • 不过其像人类一样，会自己总结和学习投篮失败或成功的经验，最后达到高命中率
    • Google的alpha 狗，就是应用此

  • 遗传算法，genetic algorithm

    • 类似强化学习
    • 模拟进化理论，淘汰弱者，适者生存
    • 通过淘汰机制去选择最优的设计或模型
    • 比如计算机会玩超级玛丽：最开始的马里奥一代很早就死，系统基于1代随机生成2代，保存这些代里最nb的，淘汰弱的，然后再次基于强者”繁衍和变异”，生出更强的马里奥。
    • 上述即遗传算法的基本思想
